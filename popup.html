How to use DOM Element Remover

  1. Click the DOM Element Remover icon in your browser toolbar.
  2. Add a new rule: enter the page's URL or domain and the CSS selector(s) of the element(s) you want removed.
  3. Use Use Current Page URL for auto-filling the URL.
  4. Toggle rules on/off using the switch next to each rule.
  5. Reload affected pages to see changes.
  6. Enjoy a smoother web experience!
For more help, see the README or contact @ramesh_madara
